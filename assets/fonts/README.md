# Embedded Monospace Font

The native After Effects ASCII Character plugin embeds a grayscale bitmap atlas generated from the following open-license font:

- **Font Name**: JetBrains Mono
- **Style**: Regular
- **License**: SIL Open Font License 1.1 (see [OFL.txt](OFL.txt))
- **Source**: [JetBrains/JetBrainsMono GitHub Repository](https://github.com/JetBrains/JetBrainsMono)
- **Version**: Latest Master (SIL OFL 1.1)

This font is selected for its clean legibility at small sizes and excellent glyph spacing.
