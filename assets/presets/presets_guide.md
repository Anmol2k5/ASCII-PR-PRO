# ASCII Character - Official Presets Guide

This guide details the configurations for three official style presets designed for **ASCII Character**. You can load these parameters using the interactive helper panel or configure them manually in Adobe After Effects.

---

## 1. Matrix Digital Rain
*A high-contrast hacking screen simulation featuring vertical character streams.*

### Parameter Configurations
* **Pixel Density**: `10 px`
* **Cell Aspect Ratio**: `130%` (tall vertical rectangular cells)
* **Character Scale**: `120%` (adds vertical overlap styling)
* **Spacing X**: `100%`
* **Spacing Y**: `100%`
* **Rotation**: `0°`
* **Character Set**: `Minimal` (`@#*:. `) or `Binary` (`10`)
* **Character Order**: `Dark to Light`
* **Color Mode**: `Original Source Color` (requires source footage to have green text streams) or `Monochrome`
* **Foreground Color**: `#00FF66`
* **Background Color**: `#000000` (Fully opaque black)
* **Brightness**: `+10%`
* **Contrast**: `150%` (forces bright green highlights over dark areas)
* **Gamma**: `1.20`
* **Quality Mode**: `Normal` (2x2 subpixel rendering)

## 2. Retro Terminal
*A classic phosphor CRT terminal with high-density monospaced text rendering.*

### Parameter Configurations
* **Pixel Density**: `8 px` (high density text)
* **Cell Aspect Ratio**: `90%`
* **Character Scale**: `100%` (neat pixel grid alignment)
* **Spacing X**: `100%`
* **Spacing Y**: `100%`
* **Rotation**: `0°`
* **Character Set**: `Classic` (`@%#*+=-:. `)
* **Character Order**: `Dark to Light`
* **Color Mode**: `Custom Foreground/Background`
* **Foreground Color**: `#33FF33` (Phosphor Green)
* **Background Color**: `#050A05` (Deep dark-green tint)
* **Brightness**: `0%`
* **Contrast**: `100%`
* **Gamma**: `1.00`
* **Quality Mode**: `High` (3x3 grid sampling for crisp glyph outlines)

## 3. Anaglyph ASCII
*A stylized chromatic-aberration inspired two-tone look with inverse grid rotation.*

### Parameter Configurations
* **Pixel Density**: `16 px` (chunky stylistic blocks)
* **Cell Aspect Ratio**: `100%`
* **Character Scale**: `90%`
* **Spacing X**: `110%`
* **Spacing Y**: `110%`
* **Rotation**: `15°`
* **Character Set**: `Blocks` (`#%*+=-:. `)
* **Character Order**: `Light to Dark`
* **Color Mode**: `Two Tone`
* **Foreground Color**: `#FF0055` (Vibrant Pink)
* **Background Color**: `#00FFFF` (Vibrant Cyan)
* **Brightness**: `-10%`
* **Contrast**: `120%`
* **Gamma**: `0.80`
* **Quality Mode**: `Draft` (fast nearest-neighbor rendering)
