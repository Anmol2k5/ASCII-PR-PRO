## ⚡ Quick Installation (3 Steps)

### Step 1: Locate Scripts Folder

**Windows:**
```
C:\Program Files\Adobe\Adobe After Effects [VERSION]\Support Files\Scripts\ScriptUI Panels\
```

**Mac:**
```
/Applications/Adobe After Effects [VERSION]/Scripts/ScriptUI Panels/
```

Replace `[VERSION]` with your After Effects version (e.g., "2024", "CC 2023")

### Step 2: Copy Script File

1. **Close After Effects** (if running)
2. Copy `ASCII_PRO_Effect.jsx` to the Scripts folder above
3. If the `ScriptUI Panels` folder doesn't exist, create it

### Step 3: Launch Script

1. **Open After Effects**
2. Go to **Window** menu
3. Find and click **ASCII_PRO_Effect.jsx**
4. The script panel will appear - dock it wherever you like!

## 🎯 First Time Use

### Create Your First Effect (60 seconds)

1. **Create a new composition** or open existing project
2. **Import video/image** and add to timeline
3. **Select the layer** in timeline
4. **Click "ASCII_PRO_Effect.jsx"** from Window menu
5. **Choose settings:**
   - Character Set: "Standard Symbols" (recommended)
   - Grid Size: 8-16 pixels (lower = more detail)
   - Color Mode: "Original" (to see your footage colors)
6. **Click "Create Effect"**
7. **Done!** Adjust controls on the "ASCII PRO Controls" layer

## 🔧 Alternative Installation Methods

### Method 2: Run as One-Time Script

Good if you only use it occasionally:

1. Open After Effects
2. **File** > **Scripts** > **Run Script File...**
3. Navigate to `ASCII_PRO_Effect.jsx`
4. Click Open

**Note:** You'll need to navigate to the file each time with this method.

### Method 3: Copy to User Scripts

For portable installation:

**Windows:**
```
C:\Users\[USERNAME]\AppData\Roaming\Adobe\After Effects\[VERSION]\Scripts\
```

**Mac:**
```
~/Library/Application Support/Adobe/After Effects/[VERSION]/Scripts/
```

Then access via **File** > **Scripts** menu.

## 🚨 Troubleshooting

### "Script not found in Window menu"

**Solution:**
- Make sure script is in `ScriptUI Panels` folder (not just `Scripts`)
- Restart After Effects after copying file
- Check that file is named `ASCII_PRO_Effect.jsx` (not .txt or .js)

### "Allow Scripts to Write Files and Access Network"

**Solution:**
1. Go to **Edit** > **Preferences** > **Scripting & Expressions**
2. Check "Allow Scripts to Write Files and Access Network"
3. Restart After Effects

### "Script contains errors"

**Solution:**
- Make sure you downloaded the complete file
- Check file hasn't been corrupted
- Re-download from original source
- Contact support if issue persists

### Script runs but nothing happens

**Solution:**
- Make sure you have a composition open (not project panel)
- Make sure a layer is selected
- Check layer isn't locked
- Try with a simple solid layer first
