## 🎓 How It Works

### Technical Overview:

1. **Script UI** - User selects options (character set, grid size, etc.)
2. **Control Null** - Creates a layer with all adjustable parameters
3. **Sample Layer** - Duplicate of source with preprocessing (mosaic, levels)
4. **Text Layer** - Uses expressions to sample colors and map to characters
5. **Real-time Updates** - All controls drive expressions for instant feedback

### Expression Logic:

```javascript
// Simplified version of the core algorithm
1. Divide composition into grid (based on Grid Size)
2. For each grid cell:
   - Sample the pixel color from Sample Layer
   - Calculate brightness (luminance)
   - Map brightness to character index
   - Add character to result string
3. Return complete ASCII text
```

### Performance Features:

- **Smart Sampling**: Multi-point sampling for quality (optional)
- **Optimized Math**: Minimal calculations in expressions
- **Cached Values**: Control values read once per frame
- **Scalable Grid**: Automatic adjustment for resolution
