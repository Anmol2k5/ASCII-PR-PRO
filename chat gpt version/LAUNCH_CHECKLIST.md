## Emergency Checklist (If Things Go Wrong)

### Technical Issues

**Script doesn't work:**
- [ ] Test on fresh AE installation
- [ ] Check After Effects version compatibility
- [ ] Review error messages
- [ ] Fix and push update immediately
- [ ] Notify affected customers

**Download issues:**
- [ ] Test download link
- [ ] Check file size/corruption
- [ ] Verify hosting service
- [ ] Provide alternative download method

### Business Issues

**Refund request:**
- [ ] Review refund policy
- [ ] Ask for feedback (what went wrong?)
- [ ] Process refund professionally
- [ ] Learn from the experience

**Negative review:**
- [ ] Respond professionally and quickly
- [ ] Address the specific issue
- [ ] Offer solution or refund
- [ ] Use feedback to improve

**Support overwhelmed:**
- [ ] Update FAQ to address common issues
- [ ] Create video answers to frequent questions
- [ ] Consider hiring support help
- [ ] Set better expectations on response time
