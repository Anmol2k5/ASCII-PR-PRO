# ASCII PRO Helper Panel

The `ASCII PRO.jsx` is an optional, lightweight ScriptUI helper panel for Adobe After Effects that lets you apply the native ASCII Character effect, apply presets, and reset parameters with a single click.

## Installation

1. Copy `ASCII PRO.jsx` to the After Effects ScriptUI Panels directory:
   * **Windows**: `C:\Program Files\Adobe\Adobe After Effects <Version>\Support Files\Scripts\ScriptUI Panels\`
   * **macOS**: `/Applications/Adobe After Effects <Version>/Scripts/ScriptUI Panels/`

2. Restart After Effects.

3. Open the panel from the top menu: `Window > ASCII PRO.jsx`.

## Features

* **Plugin Status**: Automatically checks if the native ASCII C++ plugin (`com.anmolsaini.ascii-character`) is correctly installed and active.
* **Apply ASCII Character**: Adds the native ASCII renderer to all selected layers.
* **Reset to Defaults**: Restores default parameters to the active effect instance.
* **Style Presets**: Apply pre-configured styles (Matrix Digital Rain, Retro Terminal, Anaglyph ASCII) directly to selected layers.
* **Create Demo Composition**: Automatically sets up a full HD comp with text and an active ASCII Character effect instance for immediate testing.
