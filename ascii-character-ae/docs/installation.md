# Installation

## Development Install

1. Build `ASCIICharacter.aex` in Release mode.
2. Create:

   ```text
   C:\Program Files\Adobe\Adobe After Effects 2026\Support Files\Plug-ins\ASCII Character
   ```

3. Copy:

   ```text
   build\Release\ASCIICharacter.aex
   ```

   into that folder.

4. Restart After Effects.
5. Apply `ASCII Character` to footage, still images, precomps, solids, or an
   adjustment layer.

## Uninstall

Close After Effects and remove:

```text
C:\Program Files\Adobe\Adobe After Effects 2026\Support Files\Plug-ins\ASCII Character\ASCIICharacter.aex
```
